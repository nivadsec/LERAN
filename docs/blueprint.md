# **App Name**: iTalk Academic

## Core Features:

- Smart Self-Assessment: AI tool that evaluates the student's strengths and weaknesses based on their input and performance data, to generate an academic success plan.
- Personalized Planning: Generates a detailed academic plan based on the self-assessment and desired goals. Plan is built interactively with student using an AI tool.
- Progress Tracking: Monitors student progress and provides visual feedback.
- Login: Login to system.
- Signup: Create an account in the system.

## Style Guidelines:

- Primary color: Dark slate blue (#3B5998) reflecting intellect and focus.
- Background color: Very light blue-gray (#F0F8FF) to maintain a calm, uncluttered feel.
- Accent color: Teal (#008080) for interactive elements and key highlights.
- Body and headline font: 'Alegreya', a serif font offering elegance for comfortable reading.
- Simple, line-based icons representing study skills, goal setting, and personal growth.
- Clean, modular layout with clear content hierarchy.
- Subtle transitions and progress animations.
